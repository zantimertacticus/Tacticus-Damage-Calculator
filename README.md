# Tacticus Damage Simulator

A fan-made combat simulator for **Warhammer 40,000: Tacticus**.

The simulator calculates deterministic and simulated combat outcomes between player characters and bosses using configurable combat statistics, including armour, pierce ratios, critical hits, blocking, terrain effects and damage variance.

---

## Features

### Combat Engine

- Player vs Boss calculations
- Boss normal attack
- Up to 3 boss abilities
- 22 supported damage types
- Armour vs Pierce calculation
- Damage variance (80%–120%)
- Per-hit critical rolls
- Per-hit block rolls
- Terrain modifiers
- Deterministic expected damage
- Monte-Carlo combat simulation

---

### Simulation

- 1–100 attack simulations
- Average damage
- Minimum damage
- Maximum damage
- Critical hit count
- Block count
- Expandable combat log
- Damage distribution graph

---

### Presets

- Save player builds
- Save bosses
- Import JSON
- Export JSON
- Local browser storage

---

### Interface

- Responsive desktop/mobile layout
- Dark Tacticus-inspired theme
- Clean results dashboard
- Boss ability tabs
- Optional combat graph
- Optional combat log

---
