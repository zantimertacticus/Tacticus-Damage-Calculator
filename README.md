# Tacticus Damage Calculator

v0.9

This version bundles the app into a single local JavaScript file:

```text
assets/js/app.js
```

That makes GitHub Pages deployment more reliable because the app no longer depends on many ES module imports.

Features:
- Player and boss editor
- Boss attack + 3 optional abilities
- Damage type pierce ratios
- Min / expected / max damage
- 1–100 roll simulations
- Chained crits
- Chained blocks
- Flat block damage reduction
- Terrain rules
- Chart.js damage distribution
- Full combat log option
- Named local presets
- Encounter/player/boss JSON import-export
