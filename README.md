# Tacticus Damage Calculator

v1.4

## New in v1.4

- Added calculation direction toggles:
  - Calculate Player → Boss
  - Calculate Boss → Player
- Both are enabled by default.
- You can now run only player damage, only boss damage, or both.

Keeps:
- Multiple traits per Player/Boss
- Trait preset application
- Gravis/Weaver labels and disclaimer
- Player second optional attack
- Named presets
- Encounter/player/boss JSON import/export
- Full combat log option
