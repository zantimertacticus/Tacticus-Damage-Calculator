# Tacticus Damage Simulator

A fan-made combat calculator and simulation tool for **Warhammer 40,000: Tacticus**.

This project runs fully in the browser and can be hosted for free on GitHub Pages.

## Current version

**v0.3** — Polished UI + first working combat engine.

## Features in this snapshot

- Responsive application layout
- Player editor
- Boss editor
- Boss attack / ability tabs
- All listed Tacticus damage types
- Armour vs pierce calculation
- Damage variance min / expected / max
- Per-hit crit checks
- Per-hit block checks
- Terrain modifiers
- 1–100 roll simulation
- Expandable combat log
- Simple text histogram
- Local preset saving
- JSON import/export

## Deployment

Open `index.html` locally, or upload the folder to GitHub Pages.
